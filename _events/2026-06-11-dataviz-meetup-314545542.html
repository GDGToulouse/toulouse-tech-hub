Le Seum-Score vous a montré la réalité brute de l'amertume européenne. Maintenant, préparez-vous à descendre d'un étage. Dans cette présentation en Seum-ception, j'utilise la technologie Closeread pour disséquer... le Closeread lui-même. Nous allons infiltrer le code source du projet primé au Toulouse DataViz 2026 pour comprendre comment un simple scroll peut manipuler la perception des données. Des déclencheurs invisibles aux zooms millimétrés, découvrez comment l'architecture d'un fichier .qmd peut créer une immersion totale. Ne cherchez pas la toupie : si le code défile, c'est que vous êtes encore dans le Closeread.

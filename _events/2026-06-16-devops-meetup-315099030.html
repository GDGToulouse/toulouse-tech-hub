Après une longue pause, le Meetup Toulouse DevOps refait surface avec une soirée de reprise.

Pas de hors-jeu pour vos tokens : sécurité IA & FinOps sous le même maillot ⚽
En foot comme en IA, un hors-jeu mal géré peut coûter le match. Ce soir, les Bleus entrent sur le terrain — mais avant ça, venez apprendre à défendre vos systèmes… et votre budget.

Seifeddin Mansri, Cloud CTO du Groupe SFEIR, enfile le maillot du Cloud pour un double talk sans temps mort :
🛡️ SAIF — Secure AI Framework Prompt injection, data poisoning, model extraction, Shadow AI… les menaces sont réelles. On voit comment structurer une posture de sécurité IA solide avec le framework de Google, des fondations jusqu'à la gouvernance.
⚡ Prompt Optimization & FinOps Un token, c'est de l'argent. La diète du prompt, le model routing, le prompt caching à −90 % de coûts : autant de tactiques pour faire jouer votre IA en première division sans exploser le salary cap.
Que vous soyez dev, archi ou tech lead, vous repartez avec des actions concrètes — et promis, on vous libère à temps pour le coup d'envoi.

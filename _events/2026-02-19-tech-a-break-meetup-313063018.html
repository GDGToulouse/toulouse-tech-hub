Du Python pour les devs Java !

Après plusieurs années passées à développer en Java, j’ai rejoint une entreprise en tant que dev Python. En discutant avec un ancien collègue à propos de mon nouveau travail, sa première question a été: “Mais comment tu fais pour faire de l’injection de dépendance en Python ?”

Je me suis rendu compte qu’en tant que dev Java, travaillant principalement avec Spring Boot, beaucoup de concepts et patterns sont utilisés et mis en place pour nous: IoC, injection de dépendances, ORM, repository pattern.
Nous allons rentrer (un peu) dans les détails sur ces patterns et les mettre en place dans un projet Python qui reprend tous les concepts familiers des devs Java ! Nous en profiterons pour présenter les spécificités de Python qui peuvent dérouter les développeurs qui arrivent depuis l'écosystème Java.

>_ Paul AMBLARD : Après plusieurs années en ESN, j'ai rejoint MeetCheck, une start-up dans le milieu de la visioconférence. Adepte du clean code et des bonnes pratiques, je cherche toujours comment améliorer la qualité de mes développements.
